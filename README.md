# YouTube Downloader Telegram Bot

Ushbu bot YouTube'dan videolarni yuklab olish imkonini beradi.

## O'rnatish va ishga tushirish

1. Ushbu loyihani kompyuteringizga yuklab oling.
2. `.env.example` faylini nusxalab, nomini `.env` ga o'zgartiring.
3. `.env` fayli ichiga BotFather'dan olgan `BOT_TOKEN` ni yozing:
   ```env
   BOT_TOKEN=SizningTokeningiz
   ```
4. Kutubxonalarni o'rnating:
   ```bash
   pip install -r requirements.txt
   ```
5. Botni ishga tushiring:
   ```bash
   python main.py
   ```
   yoki `run_bot.bat` faylini ikki marta bosing.
